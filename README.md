## 🧑‍💻 About Me

I'm a backend-focused developer with a full-stack mindset, building robust applications with **Spring Boot** on the server side and **Angular** on the client side. I care deeply about clean architecture, secure APIs, and writing code that scales.

Alongside my engineering work, I'm diving into **Machine Learning** - fascinated by how mathematical models can solve real-world problems.

```text
🔧  Crafting backends with Spring Boot & REST APIs

🤖  Exploring ML: sklearn, NumPy, Pandas
📐  Passionate about Mathematics
```
